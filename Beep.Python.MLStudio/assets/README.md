# Assets Directory

This directory should contain application icons:

## Required Files

### Windows
- `icon.ico` - Windows icon file (multiple sizes: 16x16, 32x32, 48x48, 256x256)

### macOS  
- `icon.icns` - macOS icon file

### Linux
- `icon.png` - PNG icon (256x256 or 512x512 recommended)

## How to Create Icons

1. Create a high-resolution image (512x512 or 1024x1024)
2. Use tools like:
   - **Windows**: Use online converters or GIMP to create .ico
   - **macOS**: Use `iconutil` or Image2Icon app
   - **Linux**: Just save as PNG

## Placeholder

If no icons are provided, the installer will use system defaults.
