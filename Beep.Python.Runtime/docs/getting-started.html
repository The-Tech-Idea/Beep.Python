Getting Started with Beep.Python.Runtime.PythonNet

This guide will help you get started with integrating Python into your .NET applications using Beep.Python.Runtime.PythonNet.

Note: This page is under construction. The complete documentation will include step-by-step instructions for setup and usage.

Prerequisites

  • Python 3.8 or higher installed on the system
  • .NET 6.0 or higher (.NET 7.0, 8.0, or 9.0 recommended)
  • Python.NET package (installed automatically via NuGet)

Quick Start

1. Install the Package

Package Manager Console
Install-Package Beep.Python.Runtime.PythonNet

2. Basic Setup

C#
// Create required services
var beepService = new BeepService();

// Initialize the runtime manager
var runtimeManager = new PythonNetRunTimeManager(beepService);

// Scan for Python installations
runtimeManager.RefreshPythonInstalltions();
var pythonRuntime = runtimeManager.PythonInstallations.FirstOrDefault();

// Initialize the Python environment
runtimeManager.Initialize(pythonRuntime, null, "default-env", PythonEngineMode.SingleUser);

3. Execute Python Code

C#
// Get a session
var session = runtimeManager.GetOrCreateDefaultSession();

// Execute Python code
var executeManager = runtimeManager.ExecuteManager;
var result = await executeManager.ExecuteCodeAsync(
    @"
import sys
print(f'Python {sys.version} on {sys.platform}')

# Calculate something
result = sum(range(1, 101))
print(f'The sum of numbers from 1 to 100 is {result}')
",
    session);

Console.WriteLine(result.Output);

4. Clean Up

C#
// Shut down the Python engine when done
runtimeManager.ShutDown();
