PythonPlotManager

Advanced data visualization and plotting capabilities for Python.NET integration

Overview

The PythonPlotManager class provides comprehensive data visualization capabilities within the Beep.Python.Runtime framework. It integrates matplotlib, seaborn, and plotly to create professional-quality charts, plots, and interactive visualizations from .NET applications.

?? Key Features
  • Support for matplotlib, seaborn, and plotly visualization libraries
  • Wide range of chart types (line, scatter, bar, contour, 3D, etc.)
  • Advanced contour plotting for scientific data visualization
  • Interactive plotting capabilities with plotly integration
  • High-resolution image export with customizable formats
  • Statistical plotting and data exploration visualizations

Helper Methods and Utilities

Data Preparation Utilities

// Generate 2D data matrix for contour plots
private double[,] GenerateZMatrix(int rows, int cols)
{
    var z = new double[rows, cols];
    for (int i = 0; i < rows; i++)
    {
        for (int j = 0; j < cols; j++)
        {
            // Example: Gaussian surface
            double x = (i - rows/2.0) / (rows/4.0);
            double y = (j - cols/2.0) / (cols/4.0);
            z[i, j] = Math.Exp(-(x*x + y*y));
        }
    }
    return z;
}

// Flatten 2D array for certain plot types
private double[] FlattenZArray(double[,] z)
{
    int width = z.GetLength(0);
    int height = z.GetLength(1);
    double[] z1D = new double[width * height];
    
    for (int x = 0; x < width; x++)
    {
        for (int y = 0; y < height; y++)
        {
            z1D[x * height + y] = z[x, y];
        }
    }
    return z1D;
}

// Serialize data for plotly JSON consumption
public string SerializeZDataForPlotly(double?[,] zData)
{
    var settings = new JsonSerializerSettings
    {
        NullValueHandling = NullValueHandling.Include,
        Converters = new List { new StringEnumConverter() }
    };
    return JsonConvert.SerializeObject(zData, settings);
}

// Helper methods for data generation
private double[] GenerateLinearRange(double start, double end, int count)
{
    double[] range = new double[count];
    double step = (end - start) / (count - 1);
    for (int i = 0; i < count; i++)
    {
        range[i] = start + i * step;
    }
    return range;
}

private double[] GenerateTemperatureField(int width, int height)
{
    var random = new Random(42); // Fixed seed for reproducible results
    double[] field = new double[width * height];
    
    for (int i = 0; i < width; i++)
    {
        for (int j = 0; j < height; j++)
        {
            double x = (i - width/2.0) / (width/4.0);
            double y = (j - height/2.0) / (height/4.0);
            
            // Simulate temperature with multiple heat sources
            double temp = 20 + // Base temperature
                         10 * Math.Exp(-((x-1)*(x-1) + (y-1)*(y-1))/2) + // Hot spot 1
                         8 * Math.Exp(-((x+1)*(x+1) + (y+1)*(y+1))/3) +   // Hot spot 2
                         random.NextDouble() * 3; // Random variation
            
            field[i * height + j] = temp;
        }
    }
    return field;
}

Best Practices

?? Chart Design

  • Choose appropriate chart types: Match visualization to data characteristics
  • Use clear labels: Provide descriptive titles and axis labels
  • Consider color schemes: Use colorblind-friendly palettes
  • Optimize resolution: Set appropriate DPI for intended output

?? Visual Aesthetics

  • Maintain consistency: Use consistent styling across related charts
  • Minimize clutter: Remove unnecessary visual elements
  • Use white space effectively: Give charts room to breathe
  • Consider your audience: Adapt complexity to viewer expertise

? Performance

  • Optimize data size: Sample large datasets for visualization
  • Use appropriate formats: PNG for detailed plots, SVG for scalable graphics
  • Cache static plots: Avoid regenerating unchanged visualizations
  • Consider interactive alternatives: Use plotly for exploratory analysis

?? Scientific Visualization

  • Include scale information: Provide color bars and legends
  • Document data sources: Include metadata about data origin
  • Use logarithmic scales when appropriate: For wide dynamic ranges
  • Validate visual accuracy: Ensure plots accurately represent data

Chart Type Selection Guide

?? Trend Analysis

Line Plot: Time series data, trends over time

?? Categorical Data

Bar Chart: Comparing categories

?? Relationships

?? Distributions

Error Handling

Common Error Scenarios

try
{
    // Check if plot manager is initialized
    if (!plotManager.IsInitialized)
    {
        Console.WriteLine("Plot manager not initialized");
        return;
    }
    
    // Validate data before plotting
    if (xData == null || yData == null)
    {
        throw new ArgumentNullException("Plot data cannot be null");
    }
    
    if (xData.Length != yData.Length)
    {
        throw new ArgumentException("X and Y data must have same length");
    }
    
    // Check for invalid values
    if (xData.Any(double.IsNaN) || yData.Any(double.IsNaN))
    {
        Console.WriteLine("Warning: Data contains NaN values");
        // Clean data or handle appropriately
    }
    
    // Create plot with error handling
    plotManager.CreatePyPlotChart(
        outputPath, xData, yData, title, xLabel, yLabel, 
        ChartType.LinePlot, 800, 600);
        
    Console.WriteLine("Plot created successfully");
}
catch (ArgumentException ex)
{
    Console.WriteLine($"Invalid arguments: {ex.Message}");
}
catch (FileNotFoundException ex)
{
    Console.WriteLine($"Output directory not found: {ex.Message}");
}
catch (Exception ex)
{
    Console.WriteLine($"Plotting error: {ex.Message}");
}
