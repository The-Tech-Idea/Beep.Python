Beep.Python.Runtime.PythonNet

Comprehensive .NET library for Python integration with runtime management, session isolation, and code execution

Overview

Beep.Python.Runtime.PythonNet is a enterprise-grade .NET library that provides seamless Python integration for .NET 6, 7, 8, and 9 applications. It offers comprehensive Python runtime management, session isolation, code execution, and virtual environment support through a robust, thread-safe architecture.

Built with scalability and enterprise requirements in mind, the library supports multi-user scenarios, async operations, package management, and provides extensive error handling and logging capabilities.

Basic Usage

Here's a simple example to get you started with Python.NET integration:

Basic Python Code Execution

using Beep.Python.RuntimeEngine;
using TheTechIdea.Beep.Container.Services;

// Initialize the runtime manager
var beepService = new BeepService();
var runtimeManager = new PythonNetRunTimeManager(beepService);

// Scan for Python installations
runtimeManager.RefreshPythonInstalltions();
var pythonRuntime = runtimeManager.PythonInstallations.FirstOrDefault();

if (pythonRuntime != null)
{
    // Initialize Python engine
    bool initialized = runtimeManager.Initialize(
        pythonRuntime, 
        null, 
        "default-env", 
        PythonEngineMode.SingleUser);

    if (initialized)
    {
        // Get a session and execute Python code
        var session = runtimeManager.GetOrCreateDefaultSession();
        var executeManager = runtimeManager.ExecuteManager;

        var result = await executeManager.ExecuteCodeAsync(
            @"
import sys
import os
print(f'Python {sys.version} running on {sys.platform}')

# Perform calculations
numbers = [1, 2, 3, 4, 5]
result = sum(x * x for x in numbers)
print(f'Sum of squares: {result}')
", 
            session);

        if (result.Success)
        {
            Console.WriteLine("Python Output:");
            Console.WriteLine(result.Output);
        }
        else
        {
            Console.WriteLine("Execution failed:");
            Console.WriteLine(result.Output);
        }

        // Clean up
        runtimeManager.ShutDown();
    }
}
