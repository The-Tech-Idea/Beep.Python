Complete documentation for all classes, interfaces, and enums in Beep.Python.Runtime.PythonNet

Overview

This page provides comprehensive API documentation for all public classes, interfaces, and enums in the Beep.Python.Runtime.PythonNet library. The architecture is designed around a multi-layered approach with dedicated managers for different aspects of Python integration.

Core Architecture

??? Runtime Layer

Manages Python.NET initialization, environment setup, and session coordination

  • PythonNetRunTimeManager
  • PythonSessionManager

? Execution Layer

Handles code execution, output capture, and error management

  • PythonCodeExecuteManager

?? Environment Layer

Manages virtual environments and package installations

  • PythonVirtualEnvManager
  • PythonPackageManager

?? Specialized Managers

Domain-specific functionality for ML, data analysis, and visualization

  • PythonMLManager
  • PythonPandasManager
  • PythonPlotManager

Key Methods:

  • Initialize() - Runtime initialization
  • ShutDown() - Clean shutdown
  • CreateScope() - Session scope management
  • RefreshPythonInstalltions() - Discovery

IPythonSessionManager

  • TerminateSession() - Session cleanup

Key Enumerations

PythonEngineMode

Defines the operational mode of the Python engine

  • SingleUser - Single user with one scope
  • MultiUserWithEnvAndScopeAndSession - Multi-user with isolation

PythonBinary

Specifies the Python distribution type

  • Pip - Standard Python with pip
  • Conda - Anaconda/Miniconda distribution

PythonSessionStatus

Represents the current status of a Python session

PythonEnvironmentType

Common Usage Patterns

?? Basic Runtime Setup

// Initialize and configure Python runtime
var runtimeManager = new PythonNetRunTimeManager(beepService);
runtimeManager.RefreshPythonInstalltions();
var pythonRuntime = runtimeManager.PythonInstallations.FirstOrDefault();
var initialized = runtimeManager.Initialize(pythonRuntime, null, "default-env", PythonEngineMode.SingleUser);

? Code Execution

// Execute Python code with session isolation
var session = runtimeManager.GetOrCreateDefaultSession();
var executeManager = runtimeManager.ExecuteManager;
var result = await executeManager.ExecuteCodeAsync("print('Hello Python!')", session);

?? Environment Management

// Create and manage virtual environments
var envManager = runtimeManager.VirtualEnvmanager;
var newEnv = new PythonVirtualEnvironment { Name = "ml-env", Path = envPath };
var created = envManager.CreateVirtualEnvironment(pythonRuntime, newEnv);

?? Package Management

// Install and manage Python packages
var packageManager = new PythonPackageManager(beepService, runtimeManager, envManager, environment, session);
var installed = await packageManager.InstallNewPackageAsync("numpy");
