? Quick Start

Get up and running with PythonPandasManager in minutes

1. Initialize and Configure Session

// Initialize the pandas manager with session management
var pandasManager = new PythonPandasManager(beepService, runtimeManager, executeManager);

// Configure for a specific user and environment
pandasManager.ConfigureSessionForUser("john.doe", "data-analysis-env");

// Or configure with existing session and environment
pandasManager.ConfigureSession(existingSession, virtualEnvironment);

2. Load and Analyze Data

// Load data from various sources
pandasManager.ReadCsv("sales_data", @"C:\Data\sales.csv");
pandasManager.ReadExcel("products", @"C:\Data\products.xlsx");
pandasManager.ReadSql("customers", "SELECT * FROM customers", connectionString);

// Perform analysis
string summary = pandasManager.Describe("sales_data");
string correlation = pandasManager.Correlation("sales_data");
pandasManager.GroupBy("sales_data", "sales_by_region", "region", "sum");

3. Transform and Export Results

// Data transformation
pandasManager.FilterRows("sales_data", "high_value_sales", "amount > 1000");
pandasManager.AddColumn("sales_data", "profit", "revenue - cost");
pandasManager.PivotTable("sales_data", "sales_pivot", "month", "region", "amount");

// Export results
pandasManager.ToCsv("sales_pivot", @"C:\Output\analysis_results.csv");
pandasManager.ToExcel("high_value_sales", @"C:\Output\high_value_analysis.xlsx");

?? Core Components

PythonPandasManager

The main class providing comprehensive pandas DataFrame operations with enterprise session management and virtual environment support.

  • 40+ DataFrame Operations: Complete pandas functionality exposure
  • Session-First Architecture: Multi-user isolation and resource management
  • Memory Management: Automatic cleanup and optimization utilities
  • Thread-Safe Operations: Concurrent access support with proper locking

IPythonPandasManager Interface

Comprehensive interface defining all pandas operations for dependency injection and testing.

  • Complete API Contract: All DataFrame operations defined
  • Async Support: Key operations available in async variants
  • Resource Management: IDisposable implementation for cleanup
