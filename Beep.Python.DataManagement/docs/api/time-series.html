Time Series Operations

Comprehensive time series analysis, DateTime operations, resampling, and temporal data processing capabilities for pandas DataFrames

?? Overview

The Time Series Operations module provides powerful functionality for working with temporal data in pandas DataFrames. From basic DateTime conversions to advanced resampling and rolling window statistics, these methods enable comprehensive time series analysis for business intelligence, financial data, and IoT applications.

Key Capabilities
  • DateTime Conversion: Parse and convert date/time columns with format validation
  • Resampling: Aggregate time series data to different frequencies (hourly, daily, monthly)
  • Rolling Windows: Calculate moving statistics for trend analysis
  • Temporal Filtering: Filter data by date ranges and time periods
  • Seasonal Analysis: Extract seasonal patterns and decompose time series
  • Performance Optimized: Efficient operations for large temporal datasets

ConvertToDateTime

Converts string or numeric columns to pandas DateTime objects with format validation and error handling.

public bool ConvertToDateTime(string dataFrameName, string columnName, string format = null, bool errors = false)

Parameters

Parameter Type Description
dataFrameName string Name of the DataFrame containing the column to convert
columnName string Name of the column to convert to DateTime
format string Optional DateTime format string (e.g., "%Y-%m-%d %H:%M:%S")
errors bool If true, raises exceptions on conversion errors; if false, sets invalid dates to NaT

Returns

bool: True if conversion successful, false otherwise

Basic DateTime Conversion

// Convert string dates to DateTime objects
pandasManager.ConvertToDateTime("sales_data", "order_date");

// Convert with specific format
pandasManager.ConvertToDateTime("transactions", "timestamp", "%Y-%m-%d %H:%M:%S");

// Convert with error handling
pandasManager.ConvertToDateTime("log_data", "event_time", null, true);

Advanced DateTime Parsing

// Load CSV with date columns
pandasManager.ReadCsv("financial_data", @"C:\Data\stock_prices.csv");

// Convert multiple date columns
pandasManager.ConvertToDateTime("financial_data", "trade_date", "%Y-%m-%d");
pandasManager.ConvertToDateTime("financial_data", "settlement_date", "%Y-%m-%d");

// Verify conversion
string info = pandasManager.GetDataFrameInfo("financial_data");
Console.WriteLine($"DataFrame Info:\n{info}");

Resample

Resamples time series data to different frequencies with various aggregation functions for temporal analysis.

public bool Resample(string dataFrameName, string outputName, string rule, string aggFunc = "mean")

Parameters

Parameter Type Description
dataFrameName string Name of the source DataFrame with DateTime index
outputName string Name for the resampled DataFrame
rule string Resampling frequency ('D'=daily, 'H'=hourly, 'M'=monthly, 'Q'=quarterly)
aggFunc string Aggregation function: 'mean', 'sum', 'count', 'min', 'max', 'std'

Common Resampling Rules

  • 'T' or 'min': Minute frequency
  • 'H': Hourly frequency
  • 'D': Daily frequency
  • 'W': Weekly frequency
  • 'M': Month end frequency
  • 'Q': Quarter end frequency
  • 'A' or 'Y': Year end frequency

Financial Data Resampling

// Load minute-level stock data
pandasManager.ReadCsv("stock_prices", @"C:\Data\minute_prices.csv");
pandasManager.ConvertToDateTime("stock_prices", "timestamp");

// Set DateTime as index for resampling
pandasManager.SetIndex("stock_prices", "timestamp");

// Resample to different frequencies
pandasManager.Resample("stock_prices", "hourly_prices", "H", "mean");
pandasManager.Resample("stock_prices", "daily_prices", "D", "ohlc");
pandasManager.Resample("stock_prices", "weekly_volume", "W", "sum");

IoT Sensor Data Aggregation

// Load high-frequency sensor data
pandasManager.ReadCsv("sensor_data", @"C:\IoT\temperature_readings.csv");
pandasManager.ConvertToDateTime("sensor_data", "reading_time");
pandasManager.SetIndex("sensor_data", "reading_time");

// Create multiple aggregations

pandasManager.Resample("sensor_data", "daily_max", "D", "max");

// Export aggregated data
pandasManager.ToCsv("hourly_avg", @"C:\Reports\hourly_temperature.csv");

RollingWindow

Calculates rolling window statistics for trend analysis and smoothing of time series data.

public bool RollingWindow(string dataFrameName, string outputName, int window, string function = "mean")

Parameters

Parameter Type Description
dataFrameName string Name of the source DataFrame
outputName string Name for the DataFrame with rolling statistics
window int Size of the rolling window (number of periods)
function string Rolling function: 'mean', 'sum', 'std', 'var', 'min', 'max', 'median'

Moving Averages for Financial Analysis

// Load stock price data
pandasManager.ReadCsv("stock_data", @"C:\Data\daily_prices.csv");
pandasManager.ConvertToDateTime("stock_data", "date");

// Calculate moving averages
pandasManager.RollingWindow("stock_data", "ma_5", 5, "mean");     // 5-day MA
pandasManager.RollingWindow("stock_data", "ma_20", 20, "mean");   // 20-day MA
pandasManager.RollingWindow("stock_data", "ma_50", 50, "mean");   // 50-day MA

// Calculate rolling volatility
pandasManager.RollingWindow("stock_data", "volatility", 20, "std");

// Generate trading signals
pandasManager.AddColumn("stock_data", "signal", 
    "np.where(ma_5 > ma_20, 'BUY', 'SELL')");

Sales Trend Analysis

// Load daily sales data
pandasManager.ReadCsv("sales_daily", @"C:\Data\daily_sales.csv");
pandasManager.ConvertToDateTime("sales_daily", "sale_date");

// Calculate rolling statistics for trend analysis
pandasManager.RollingWindow("sales_daily", "sales_7day_avg", 7, "mean");
pandasManager.RollingWindow("sales_daily", "sales_7day_sum", 7, "sum");
pandasManager.RollingWindow("sales_daily", "sales_30day_max", 30, "max");

// Detect anomalies using rolling standard deviation
pandasManager.RollingWindow("sales_daily", "rolling_std", 14, "std");
pandasManager.AddColumn("sales_daily", "z_score", 
    "(sales - sales_7day_avg) / rolling_std");
pandasManager.FilterRows("sales_daily", "anomalies", "abs(z_score) > 2");

Date Range Filtering

Filter DataFrames by specific date ranges and time periods for focused analysis.

// Filter by date range
pandasManager.FilterRows("time_series_data", "q1_data", 
    "date >= '2024-01-01' and date <= '2024-03-31'");

// Filter by specific year
pandasManager.FilterRows("historical_data", "year_2024", 
    "date.dt.year == 2024");

Seasonal Decomposition

Extract trend, seasonal, and residual components from time series data.

// Create seasonal features
pandasManager.AddColumn("sales_data", "month", "date.dt.month");
pandasManager.AddColumn("sales_data", "quarter", "date.dt.quarter");
pandasManager.AddColumn("sales_data", "day_of_week", "date.dt.dayofweek");
pandasManager.AddColumn("sales_data", "is_weekend", 
    "date.dt.dayofweek >= 5");

Time Series Aggregation

Combine resampling with grouping for multi-dimensional time series analysis.

// Group by category and resample
pandasManager.ExecutePythonCode(@"
    # Group by product category and resample daily
    daily_sales = sales_data.groupby('category').resample('D', on='date')['amount'].sum()
    daily_sales = daily_sales.reset_index()
");
pandasManager.StoreDataFrameFromPython("daily_sales_by_category", "daily_sales");

? Best Practices

Performance Tips
  • Set DateTime Index: For resampling operations, set your DateTime column as the index
  • Sort by Time: Ensure your data is sorted by timestamp for optimal performance
  • Use Appropriate Frequencies: Choose resampling frequencies that match your analysis needs
  • Memory Management: For large datasets, process in chunks or use streaming operations
Common Pitfalls
  • Timezone Awareness: Be explicit about timezones when working with global data
  • Missing Timestamps: Handle gaps in time series data appropriately
  • Irregular Frequencies: Some operations require regular time intervals
  • Index Reset: Remember to reset index after resampling if needed
