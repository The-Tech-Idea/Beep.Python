Welcome to Beep.Python Host Admin Setup

Beep.Python Host Admin is a professional Python Environment and LLM Management System that allows you to:

* Manage Python virtual environments
* Discover and download LLM models from Hugging Face
* Run local inference with GPU acceleration
* Manage RAG (Retrieval Augmented Generation) pipelines
* Role-based access control for multi-user deployments

SYSTEM REQUIREMENTS:
- Windows 10/11 (64-bit)
- 4GB RAM minimum (8GB+ recommended for LLM inference)
- 10GB free disk space (more for LLM models)
- Optional: NVIDIA GPU with CUDA support for accelerated inference

IMPORTANT: 
This application runs a local web server. Make sure no other application is using port 5000 (or configure a different port during installation).
