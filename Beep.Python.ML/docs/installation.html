Installation

?? System Requirements

.NET Requirements

  • .NET 6.0 or higher - Supports .NET 6, 7, 8, 9
  • C# 10.0+ - Modern language features
  • NuGet Package Manager - For dependency management
  • Visual Studio 2022 or VS Code (recommended)

Python Requirements

  • Python 3.8 or higher - Required for ML operations
  • pip - Python package installer
  • Virtual environments - Recommended for isolation
  • 4GB+ RAM - For ML computations

Windows Requirements

  • Windows 10/11 - x64 architecture
  • Visual C++ Redistributable - For Python.NET
  • Windows Terminal - Recommended for Python management
  • PowerShell 5.1+ - For automation scripts

macOS/Linux Requirements

  • macOS 10.15+ or Ubuntu 18.04+
  • Homebrew (macOS) or apt (Ubuntu)
  • Python development headers
  • GCC compiler - For native extensions

?? Quick Installation

Get started quickly with Beep.Python.ML using these simple commands.

1. Install NuGet Package

# Package Manager Console
Install-Package Beep.Python.ML

# .NET CLI
dotnet add package Beep.Python.ML

# Package Reference (add to .csproj)
<PackageReference Include="Beep.Python.ML" Version="1.0.0" />

2. Install Python Dependencies

# Install core ML packages
pip install pandas>=1.5.0 numpy>=1.21.0 scikit-learn>=1.2.0

# Install additional ML frameworks
pip install xgboost>=1.7.0 lightgbm>=3.3.0 matplotlib>=3.6.0 seaborn>=0.12.0

3. Verify Installation

using Beep.Python.ML;
using Beep.Python.Model;
using TheTechIdea.Beep.Container.Services;

// Quick verification
var beepService = new BeepService();
var pythonRuntime = new PythonNetRunTimeManager(beepService);
var sessionInfo = new PythonSessionInfo { Username = "test_user" };
var mlManager = new PythonMLManager(beepService, pythonRuntime, sessionInfo);

Console.WriteLine("? Beep.Python.ML installed successfully!");

??? Platform-Specific Instructions

Windows Setup

Windows-Specific Configuration

# 1. Install Visual C++ Redistributable (required for Python.NET)
# Download from: https://aka.ms/vs/17/release/vc_redist.x64.exe

# 2. Configure PATH environment variable
$env:PATH += ";C:\Python311;C:\Python311\Scripts"

# 3. Set PYTHONPATH (optional, for custom modules)
$env:PYTHONPATH = "C:\your\custom\python\modules"

# 4. Configure Python.NET environment variables
$env:PYTHONNET_PYDLL = "C:\Python311\python311.dll"

# 5. Test Python.NET integration
python -c "import sys; print(sys.version); print(sys.executable)"
Windows Common Issues
  • Python.NET DLL errors: Install Visual C++ Redistributable
  • PATH issues: Ensure Python is in system PATH
  • Permission errors: Run as administrator for system-wide installation
  • Antivirus interference: Add Python directory to antivirus exclusions

macOS Setup

macOS-Specific Configuration

# 1. Install Xcode Command Line Tools (if not already installed)
xcode-select --install

# 2. Configure shell profile
echo 'export PATH="/opt/homebrew/bin:$PATH"' >> ~/.zshrc
echo 'export PYTHONPATH="/opt/homebrew/lib/python3.11/site-packages:$PYTHONPATH"' >> ~/.zshrc
source ~/.zshrc

# 3. Install additional dependencies
brew install pkg-config libffi openssl

# 4. Configure compiler flags (if needed)
export LDFLAGS="-L$(brew --prefix openssl)/lib"
export CPPFLAGS="-I$(brew --prefix openssl)/include"

# 5. Test installation
python3 -c "import pandas, numpy, sklearn; print('? All packages imported successfully')"

Linux Setup

Linux-Specific Configuration

# 1. Install development packages
sudo apt update
sudo apt install build-essential python3-dev python3-pip python3-venv
sudo apt install libffi-dev libssl-dev libxml2-dev libxslt1-dev libjpeg8-dev zlib1g-dev

# 5. Test installation

?? Installation Troubleshooting

Common Issues
  • Python not found: Ensure Python is in system PATH
  • Permission errors: Run installer as administrator (Windows)
  • Package conflicts: Use virtual environments to isolate dependencies
  • Missing Visual C++: Install Visual C++ Redistributable (Windows)
  • SSL certificate errors: Update certificates or use --trusted-host with pip
Quick Fixes
  • Upgrade pip: python -m pip install --upgrade pip
  • Clear pip cache: pip cache purge
  • Reinstall packages: pip install --force-reinstall package_name
  • Use conda alternative: conda install package_name
